# Code of Conduct

MotionAI should be a practical, respectful open-source workspace project.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but accept correction quickly.
- Keep technical disagreement focused on evidence, tradeoffs, tests, and user impact.
- Protect user privacy and never post secrets, tokens, private data, or exploit details in public threads.
- Prefer honest status labels over hype.

## Unacceptable behavior

- Harassment, threats, hate speech, or personal attacks.
- Publishing private information without permission.
- Posting credentials, exploit details, or sensitive security reports publicly.
- Repeatedly derailing issues or PRs after maintainers ask you to stop.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block users; or restrict participation to protect the project and community.
